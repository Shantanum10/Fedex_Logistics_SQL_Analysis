# 🚚 FedEx Logistics Optimization using SQL

An end-to-end SQL analytics project that analyzes FedEx logistics operations to uncover insights into shipment delays, route efficiency, warehouse performance, delivery agent productivity, and overall supply chain performance. Using SQL, the project transforms operational data into actionable business insights that support data-driven decision-making and logistics optimization.

---

# 📊 Project Overview

This project integrates multiple logistics datasets, including:

- Shipments
- Routes
- Warehouses
- Delivery Agents
- Orders

The datasets were connected using common identifiers such as **Shipment_ID**, **Route_ID**, **Warehouse_ID**, and **Agent_ID** to create a relational database capable of answering complex business questions related to logistics performance.

# 💡 Key Business Insights

- Analyzed **1,000+ shipment records** to evaluate logistics performance.
- Identified the **Top 10 delayed delivery routes** based on average delay.
- Measured **On-Time Delivery Percentage** to assess delivery reliability.
- Ranked warehouses based on shipment processing efficiency.
- Evaluated delivery agent productivity using shipment volume and delivery performance.
- Identified **Traffic** as the leading cause of shipment delays, followed by **Weather** and **Technical Issues**.
- Compared route efficiency to identify high-performing and underperforming logistics routes.
- Generated insights to improve operational efficiency and reduce transportation delays.

---

# 📌 Business Recommendations

- Prioritize optimization of high-delay routes to improve delivery performance.
- Implement predictive delay monitoring for proactive issue resolution.
- Optimize warehouse operations to reduce shipment processing time.
- Improve route planning using historical delay analysis.
- Increase operational focus on traffic-prone regions through better scheduling.
- Monitor delivery agent performance regularly to improve service quality.
- Utilize data-driven logistics planning to reduce operational costs and improve customer satisfaction.
